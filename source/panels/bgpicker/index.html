Reload TETR.IO after adding new backgrounds!
Be sure to disable backgrounds configured in the game.
Regular custom background
Supported formats: Any image (animated formats won't play unless set by the music graph).
Video backgrounds
These backgrounds can only be used through the music graph.
Main window -> Templates -> video background setup for automatic setup
Supported formats: mp4, webm.
Animated custom background (DEPRECATED)
You can only have one animated background at a time.
Supported formats: GIF, animated PNG, animated WebP.
Animated backgrounds will be marked as animated.
DEPRECATED: Normal backgrounds will animate when used with the music graph.
